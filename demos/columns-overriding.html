Zenponsive - A responsive CSS framework
with (Zenphoto) gallery themes in mind

Demo 2: Overriding behaviour using the extra classes

This is a speciality of this framework. You can use special predefined classes to override the default behaviour. So you can choose to turn colums into different ones on the smaller resolutions:

  • .xxx-979 / .xxx-767 / .xxx-479 (where xxx is col01 to col12)
    Attach these to any column to turn it to a different colum on the specific breakpoint width.
  • .xxx-979-n / .xxx-767-n / .xxx-479-n (where xxx is col01 to col12)
    Attach these to any wrapper to turn all direct child columns into the column chosen.

Of course you need to take care that the columns chosen are even to make sense. (e.g. combining col02 with col09 in a row is obviously not).

This is also useful for galleries. Here some example with images instead of text therefore.
